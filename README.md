## Overview

This project aims to make openSUSE packages easier to use on Fedora. Most of the work is directly based on the openSUSE project. When bugs or issues are identified, the original authors are contacted â€” they are very helpful and usually fix problems quickly. In this way, the project also contributes to testing and improving the packages.

## Features

- Easy-to-use packages for Fedora derived from openSUSE
- Active collaboration with upstream openSUSE authors
- Contributes to package testing and improvement
- Support for **aarch64** architecture

## Contributing

Contributions are welcome! If you encounter bugs or issues, please report them. Collaboration with the upstream authors is highly encouraged to ensure fixes are applied quickly.
